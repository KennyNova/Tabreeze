# Tabreeze Video Messaging Lock

## Core Positioning

- Primary line: `Every new tab becomes your personal productivity dashboard.`
- Anchor claim: `Tabreeze replaces your browser new tab page with a customizable command center.`
- Value stack: `Start faster -> see what matters -> shape the layout around your workflow.`

## Audience

- Primary: frequent tab-openers who want search, bookmarks, tasks, and context in one place.
- Secondary: technical users who also want optional homelab status visibility.

## Final Feature Order (On-Camera Priority)

1. Search launcher (multi-provider + custom source support)
2. Tile layout control (add/remove/drag/resize/save)
3. Reactive presets + custom breakpoints
4. Browser-native bookmarks widget
5. Tasks + calendar + weather at-a-glance
6. Theme and wallpaper personalization
7. Optional homelab status checks

## Voiceover/Caption Safe Lines

- `One shortcut, full context: search, links, schedule, and tasks.`
- `Arrange tiles your way, then save the layout.`
- `Use reactive presets or define custom breakpoints for each screen size.`
- `Your browser bookmarks are right here, no extra sync service needed.`
- `Make it yours with themes, wallpaper, and day/night styling.`
- `For technical setups, optional homelab status is built in.`

## Locked CTA Options

- Primary CTA: `Tabreeze - Make every tab useful.`
- Alternate CTA: `Stop opening blank tabs. Start with context.`

## Messaging Guardrails (Do Not Break)

- Say `new tab dashboard` or `replaces your new tab page`.
- Do not present the product as a popup workflow.
- Describe bookmarks as browser-native (not a third-party bookmark cloud).
- Present homelab checks as optional and user-configured.
- Avoid claims that imply unrelated Google Search modification.
